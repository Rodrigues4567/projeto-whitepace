# WhitePace
O projeto Whitepace é um site institucional fictício que promove serviços de criação de websites, lojas virtuais e blogs para empresas de diversos portes.

### Tecnologias usadas:

- HTML
- CSS
- Javascript

Projeto 100% responsivo!
